## :pencil: About the repo

The purpose of this repository is to test using the StoryBook framework

### :wrench: How to use

```
  // 1 - Git Clone
  git clone https://github.com/TiagoBehenck/story-book

  // 2 - Installing the dependencies
  yarn install

  // 3 - Run the application
  yarn storybook
```
